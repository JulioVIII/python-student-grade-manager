# Python Student Grade Manager

A simple student grade manager written in Python.

## Features

- Add students
- View all students
- Search for a student
- Input validation
- Console application

## Technologies

- Python 3

## How to Run

```bash
python student_grade_manager.py
```

## Example

```text
--- STUDENT GRADE MANAGER ---
1. Add student
2. View students
3. Search student
4. Exit
```
